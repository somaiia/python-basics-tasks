# Python Basics Tasks

**Author:** Somaia Mahmoud Shapaan

A collection of Python programming exercises covering fundamentals: control flow, data structures, functions, and Jupyter notebooks.

## Structure

```
python-basics-tasks/
├── task1/
│   ├── password_checker.py     # While loop password validation (3 attempts)
│   ├── multiplication_table.py # Print the 7x multiplication table
│   └── guest_list.py           # List operations: insert, sort, remove guests
├── task2/
│   ├── products_filter.py      # Filter products above a price threshold
│   └── salary_updater.py       # Update employee salaries using a dictionary
└── task3/
    └── python_task3.ipynb      # Functions: count_vowels & find_max
```

## Topics Covered
- `while` loops and `break`
- `for` loops with f-strings
- List methods: `insert`, `append`, `sort`, `del`
- Dictionary operations and iteration
- Function definitions and return values
- Jupyter Notebook workflow
